Week Two

Webdesign

First real life skill, but wow I never imagined it to be that time consuming

This week Marcello introduced us to webdesign, just like it is done at the MIT. It felt like a rollercoster. Install that, download this, create an account here, link this and boom. Honestly, I still do not really know how my VSC is connected to github and how github works in total. But you know what they say, never change a running system.

We are supposed to document our module progress here which is a really refreshing method. It exposes you to new challenges, but we will come to that later. What fascinates me is how fast you could basically start a website, I guess there is a reason why webdeveloper jobs have been cut since the early 2000s. I mean we do it in an old school way but with tools like framer wix etc. it seems qutie easy to do this for an completly beignner, which is great for young start ups and businesses.

Challenges

This page seemed to be a lot easier than it actually was hahah. I thought you just link and implemnet some pictures aand boom it is done. but as it turnd out it is a lot of work to get an website in a formst that you really like. Thankfully we livr in 2025 and there are so many tools that help you with programming. I think without chat gpt and othrt ai tools this would have taken double the time. The biggest problems till this point where:

  • images, video format
  • File size
  • Folder and link structure
  • css js implementations
  • datatransfer
